Доукментация Swagger доступна по адресу:
/rest/swagger-ui.html

Список методов API:

GET /rest/transfers
Возвращает список всех переводов.

POST /rest/transfers
Возвращает отфильтрованный список переводов.
Параметры тела запроса (XML):
sender - отправитель перевода
receiver - получатель перевода
Если один из параметров отсутствет в теле запроса, фильтрация по этому параметру не происходит.
По сути, аналог методов /rest/transfers/in/{to} и /rest/transfers/out/{from}, который принимает XML.

GET /rest/transfers/{id}
Возвращает информацию о переводе по идентификатору id.

POST /rest/transfers/create
Создает перевод.
Параметры тела запроса (XML):
sender - отправитель перевода
receiver - получатель перевода
amount - сумма перевода
comment - комментарий

GET /rest/transfers/in/{to}
Возвращает список переводов, полученных на карту to.

GET /rest/transfers/out/{from}
Возвращает список переводов, отправленных с карты from.
